# AI Chatbot Using Natural Language Processing

## Live Demo

https://nlp-chatbot-project-dgqzndh89wuygpyynnhtsf.streamlit.app/

## Overview

This project presents an AI chatbot that uses Natural Language Processing techniques to understand user queries and generate meaningful responses.

The chatbot uses semantic similarity and transformer-based sentence embeddings for intelligent response retrieval.

---

## Features

- Conversational chatbot interface
- Semantic similarity matching
- Transformer-based sentence embeddings
- Knowledge base retrieval
- Streamlit web interface
- Deployable web application

---

## Technologies Used

- Python
- Streamlit
- Sentence Transformers
- Scikit-learn
- NumPy

---

## System Architecture

User Input  
↓  
Text Processing  
↓  
Sentence Embedding Generation  
↓  
Semantic Similarity Calculation  
↓  
Knowledge Base Matching  
↓  
Chatbot Response
